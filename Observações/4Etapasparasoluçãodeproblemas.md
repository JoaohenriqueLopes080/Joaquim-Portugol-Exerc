# Etapas para Solução de Problemas

## 1. Identificar a Entrada
- Verifique como os dados serão fornecidos.
- Determine de onde os dados serão lidos: **teclado** ou **arquivo**?

## 2. Identificar a Saída
- Defina quais informações devem ser fornecidas.
- Determine para quem essas informações serão destinadas.

## 3. Entender o Problema
- Compreenda o problema completamente.
- Elabore uma solução apropriada com base no entendimento.

## 4. Encontrar a Melhor Estratégia de Solução
- Após identificar o problema, escolha a melhor forma de implementá-lo.
- Avalie a estratégia mais eficiente e eficaz para resolver o problema.
